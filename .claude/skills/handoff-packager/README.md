# Handoff Packager

Alle Artefakte → Cover-Sheet mit Links, nächsten Schritten, und expliziter PM-vs-Dev-Trennung.

Kein neues Artefakt — bündelt was bereits existiert.

## Input
Prüft automatisch welche Artefakte im aktuellen Run vorhanden sind.
Pflicht: `spec.md` + `eval.md`. Alles andere optional.

## Invocation
`/handoff-packager`

## Output
`output/YYYY-MM-DD/handoff/handoff.md`
