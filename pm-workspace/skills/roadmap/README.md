# Roadmap

Noch keine Skills in diesem Bereich.

## Erweiterungspunkt

Roadmap-Skills sind der natürliche nächste Schritt nach der Strategy-Phase.
Mögliche Skills:
- `roadmap-planner` — Decision Brief → priorisierte Roadmap mit Quartals-Planung
- `release-noter` — Fertige Features → Release Notes

## Plugin-Kandidaten

Roadmap-Skills profitieren besonders von Tool-Integrationen:
- **Linear** — Issues und Milestones direkt schreiben
- **Jira** — Tickets anlegen aus der Spec
- **Notion** — Roadmap-Seiten befüllen

## Wie du einen neuen Skill hinzufügst

1. Unterordner anlegen: `skills/roadmap/[skill-name]/`
2. `SKILL.md` schreiben (Frontmatter + Beschreibung + Input/Output)
3. Diese README.md updaten
4. `skills/README.md` updaten
