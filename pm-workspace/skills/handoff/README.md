# Handoff

Alles was Dev braucht, gebündelt. Kein Copy-Paste — nur Verweise.

## Skills

| Skill | Beschreibung |
|-------|-------------|
| [handoff-packager](handoff-packager/SKILL.md) | Spec + Eval + Prototyp + Decision Brief → Cover-Sheet mit Links und nächsten Schritten |

## Typischer Flow

```
output/spec-eval/spec.md
output/spec-eval/eval.md
output/prototyping/option-comparison.md   (optional)
output/strategy/decision-brief.md         (optional)
    ↓
/handoff-packager
    → output/handoff/handoff.md
```

## Was im Handoff steht

- Feature-Name + 1-Satz-Zusammenfassung
- Links zu allen Artefakten (keine Kopien)
- Konkrete nächste Schritte (3-5 Punkte)
- Trennung: was PM evaluiert hat vs. was Dev noch prüfen soll
- Offene Fragen und Risiken aus dem Decision Brief
