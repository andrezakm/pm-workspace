# Option Stormer

Spec → 3 strukturell verschiedene Prototypen + Vergleichstabelle.

Nicht eine Lösung und hoffen — drei Richtungen, dann entscheiden. Fällt eine Variante durch die Eval, lohnt sich trotzdem die Frage: welche Prinzipien funktionieren? Rekombination führt oft zu besseren Ergebnissen als Iteration.

## Input
`output/YYYY-MM-DD/spec-eval/spec.md`

## Invocation
`/option-stormer`

## Output
- `output/YYYY-MM-DD/prototyping/option-a/app.py`
- `output/YYYY-MM-DD/prototyping/option-b/app.py`
- `output/YYYY-MM-DD/prototyping/option-c/app.py`
- `output/YYYY-MM-DD/prototyping/option-comparison.md`
