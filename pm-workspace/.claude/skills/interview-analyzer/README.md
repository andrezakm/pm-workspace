# Interview Analyzer

Interview-Transkripte → Einzel-Analysen (6 Dimensionen) + Cross-Interview-Synthese.

## Input
Ordner mit Interview-.md-Dateien

## Invocation
`/interview-analyzer input/interviews/`

## Output
- `output/YYYY-MM-DD/discovery/interview-individual-[nachname].md` (eine pro Interview)
- `output/YYYY-MM-DD/discovery/interview-synthesis.md`
